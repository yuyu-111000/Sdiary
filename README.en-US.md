## ‚ú® Features

- Minimalist UI design
- Markdown real-time rendering
- Cross-platform (Android, Windows, macOS, Web, Linux)
- Responsive layout for multiple screen sizes
- All data stored locally ‚Äî no internet required
- Privacy mode to protect your secret entries
- Supports images, audio, and video
- Supports math formulas, mind maps, charts, flowcharts, Gantt charts, sequence diagrams, and sheet music
- Multi-language support
- Dark mode support
- Tag-based classification
- Weather, mood, and location tracking
- Export to multiple file formats
- Share as text or image
- WebDAV backup support
- LAN sync transfer
